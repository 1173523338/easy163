# easy163
安卓版网易云助手，解锁灰色和收费歌曲

预览版已发布

已知问题：
1. 偶尔不稳定
2. 暂不支持下载和收藏
3. 音源待改进

使用方式：
开启本软件后，设置系统代理为：

主机：127.0.0.1

端口：8080

如 MIUI：

WLAN -> 连接的 WLAN -> 代理 -> 手动

开启本软件后如遇到设备网络异常请取消代理设置并关闭本软件
